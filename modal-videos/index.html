Back and Forth

This module is meant to take rows of content (images and text) and it will display them with the image taking up 1/3rd of the row and the content filling the rest.

Then, just for fun, it’ll automatically reverse the order of the next row. Isn’t that grand?

This content can have a title, or not, and you can include a button below it, or create some inline links using the editor. Or both! The world is your oyster, so get shucking.

This row is reversed!

Everything is set up the same in the back-end, but we’ll make sure it reverses nice and automatically, because that’s just so much nicer than fiddling with things. And when it needs to break down for mobile, we can keep the order consistent so the experience is nicer for the user.

When I was a kid, I wanted to be a magician. I guess this is as close as I’ve gotten.

The image can also be a video. The video will play in a lightbox when it’s clicked. All of this happens automatically when you just paste a URL to YouTube (or Vimeo) into the backend there. Truly, we’re living in a time of wonder and possibility.
